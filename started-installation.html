Установка

Подключение CSS

Сначала включите файл CSS в HTML head:

<link rel="stylesheet" href="css/style.css">

Подключение JS

Подключите скрипты в указанном порядке перед закрывающим тегом body

<script src="js/jquery-3.6.0.min.js"></script>
<script src="js/script.js"></script>
<script>
  $(function(){
    $('.wrapper').pluginSlider({});
  })
  <script>

Структура HTML

Вам нужна специальная разметка.

<!-- Set up your HTML -->
<div class="wrapper">
  <div class="slider-container">
    <div class="slider-track">
      <div class="slider-item"></div>   
    </div>
  </div>
  <div class="slider-buttons">
    <button class="btn-prev">Prev</button>
    <button class="btn-next">Next</button>
  </div>
</div>

Содержимое слайдеров находиться внутри HTML разметки

<div class="slider-item"></div>

Кол-во слайдов не ограничено
